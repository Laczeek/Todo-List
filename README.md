Simple web application, customisable and saved in local storage.

Hosted on Vercel
Link: todo-list-gamma-ten-13.vercel.app
# Features
- Dark mode
- Drag n drop
- Every action is saved locally
- Alert on almost every action
- Every task item is customisable

- Vite
- React
- Redux toolkit
- DnD kit
- uuidv4
